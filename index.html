Meus Projetos

  • Churrascômetro

    Calculadora de Churrasco

    Descrição:

    A "Calculadora de Churrasco" é uma aplicação simples e prática para auxiliar no planejamento de churrascos e festas. Ela permite que você insira algumas informações básicas, como a duração do churrasco, o número de adultos e crianças que estarão presentes, e com base nesses dados, calcula a quantidade estimada de carne e bebida necessária para a sua festa.

    Como Usar:

    1. Abra a aplicação "Calculadora de Churrasco".
    2. Insira a duração estimada do churrasco em horas.
    3. Informe o número de adultos e crianças que estarão presentes.
    4. Clique no botão "Calcular".
    5. A aplicação exibirá os resultados com a quantidade estimada de carne e bebida necessária.

    Observações:

    • Os cálculos são aproximados e podem variar dependendo das preferências dos seus convidados.
    • Lembre-se de considerar a variedade de carnes (por exemplo, linguiça, frango, carne bovina) e tipos de bebida (por exemplo, refrigerantes, cerveja, água) ao fazer as compras.
    • A aplicação é destinada apenas para fins de planejamento e estimativa. Certifique-se de ajustar as quantidades com base nas suas necessidades específicas.

    Tecnologias Utilizadas:

    • Linguagem de Programação: JavaScript, CSS e HTML


  • Página de Captura

    Página de Captura

    Linguagens Utilizadas:

    • HTML
    • CSS

    Objetivo:

    O objetivo principal desta página é capturar informações de contato de visitantes interessados. Geralmente, é usada para construir uma lista de e-mails para futuras comunicações de marketing, newsletters, anúncios, entre outros. A página fornece um formulário de inscrição onde os visitantes podem inserir seu nome e e-mail.

    Características Principais:

    • Formulário de Inscrição: A página possui um formulário simples onde os visitantes podem inserir seu nome e endereço de e-mail.
    • Design Atraente: O design da página é limpo e atraente, com elementos de design que incentivam os visitantes a preencher o formulário.
    • Integração com JavaScript: JavaScript é usado para validar os campos do formulário e fornecer uma experiência interativa.
    • Responsividade: A página é responsiva, o que significa que se adapta a diferentes tamanhos de tela, proporcionando uma experiência de usuário consistente em dispositivos móveis e desktop.

    Uso Potencial:

    Esta página de captura pode ser usada por empresas, blogueiros, profissionais de marketing digital e qualquer pessoa que deseje construir uma lista de contatos para fins de comunicação futura. É comumente usado para promover produtos, serviços, conteúdo informativo ou eventos.
